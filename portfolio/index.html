Portfolio

Locale — Hyper-local GPS aware web app

Locale is a hyper-local GPS aware social web application showing you local Instagram photos, Foursquare check-ins, Twitter tweets, and GPS tagged message posts. It is fully responsive and can be used on most modern mobile browsers.

Status: Live

URL: locale.herokuapp.com

Technologies: Rails, AngularJS, Omniauth, LeafletJS, Tuktuk RWD, APIs

Team Members: Brian Schroer and Xiao G. Wu

Github: xiaogwu/locale

CodeClips — Short Q&A video clips

CodeClips is a question and answer site similar to Stack Overflow where professionals and enthusiast can ask questions and other members can respond with short video clips to facilitate collaborative knowledge sharing. Questions and answers can be up or down voted, profile pages showcasing answers and lessons.

Status: Live

URL: codeclips.herokuapp.com

Technologies: Rails, Omniauth, Devise, Youtube API

Team Members: Alex Altair, Dave Ferris, Ryan Waters and Xiao G. Wu

Github: xiaogwu/codeclips
